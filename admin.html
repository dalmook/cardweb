음성 업로드

선택한 MP3/M4A를 GitHub 저장소의 audio 폴더에 올리고, GitHub Actions로 문제 음성 목록을 갱신합니다.

홈으로

업로드 설정

토큰은 코드에 저장되지 않으며 현재 브라우저 세션에서만 보관됩니다.

저장 경로: audio/{토픽폴더명}/{파일명}.mp3 또는 .m4a

삭제 관리

토픽 폴더의 음성 파일을 조회한 뒤 선택 삭제하거나, 토픽 폴더 안의 음성 파일을 모두 삭제합니다.

조회된 파일이 없습니다.
